# NotNullReflexion

Proxy for System.Reflection with no null references whatsoever.

## Supported types

`System.Reflexion.Assembly`
`System.Reflexion.AssemblyName`

## Supported frameworks

+ .Net Standard 2.1
+ .Net Core 3.1
+ .Net Framework 4.8
+ .Net 5.0
+ .Net 6.0

64-bits platform only, nullable enabled, C# 10.
